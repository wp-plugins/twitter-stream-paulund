Adds a Twitter stream to your Wordpress widget

== Description ==

This Wordpress plugin will add your Twitter stream widget to your Wordpress blog.

All you have to do is add your username and the amount of tweets you want to display.

This will also display a Twitter follow button for your visitors to follow.

This plugin uses the Twitter API to get the latest tweets and will use the Wordpress transient API to store your tweets every 10 minutes.


== Installation ==

1. Upload `paulund-google-badge-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
